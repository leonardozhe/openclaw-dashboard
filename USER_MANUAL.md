# MeetClaw 桌面应用用户手册

欢迎使用 MeetClaw 桌面应用！本手册将指导您如何使用这个强大的 OpenClaw 管理工具。

## 概述

MeetClaw 是一个基于 Tauri 框架的跨平台桌面应用程序，旨在为您提供一个美观、高效的方式来管理和监控 OpenClaw 系统。该应用结合了现代 Web 技术的美观性和原生桌面应用的性能优势。

## 安装

### Windows
1. 下载 `.msi` 或 `.exe` 安装文件
2. 双击运行安装程序
3. 按照安装向导完成安装过程

### macOS
1. 下载 `.dmg` 文件
2. 双击打开磁盘映像
3. 将 MeetClaw 拖拽到 Applications 文件夹
4. 从 Launchpad 启动应用

### Linux
1. 下载适合您发行版的包（`.deb` 或 `.AppImage`）
2. 安装或运行该文件
3. 首次运行可能需要设置执行权限

## 主要功能

### 1. 系统状态监控
- **OpenClaw 状态卡片**: 实时显示 OpenClaw 系统的运行状态
- **设备监控**: 监控本地设备和连接状态
- **系统资源**: 显示内存使用情况和系统性能指标

### 2. 终端界面
- **TUI 终端**: 内置的美观终端界面，支持命令执行
- **快速命令按钮**: 一键执行常用命令
- **实时输出**: 命令执行结果实时显示
- **语法高亮**: 不同类型输出使用不同颜色显示

### 3. 聊天输入
- **语音动画**: 精美的波浪形动画效果
- **智能提示**: 支持命令自动补全和提示

### 4. 通知系统
- **系统通知**: 重要事件的通知提醒
- **状态变更**: 系统状态变化的实时反馈

## 使用场景

### 作为 CLI 替代品
MeetClaw 桌面应用可以直接执行 OpenClaw CLI 命令，而无需通过网络接口。这解决了以下问题：
- 浏览器安全策略限制
- CORS 问题
- 网络延迟
- 权限验证问题

### 日常监控
使用桌面应用可以：
- 实时监控 OpenClaw 系统状态
- 快速执行诊断命令
- 观察系统性能指标
- 接收系统事件通知

### 本地开发
对于开发者，桌面应用提供了：
- 与本地 OpenClaw 实例的无缝集成
- 原生命令执行能力
- 丰富的调试和监控工具

## 界面介绍

### 顶部状态栏
- **OpenClaw 状态**: 显示 OpenClaw 服务的运行状态
- **设备状态**: 显示本地设备的连接和运行状况
- **系统资源**: 显示内存使用情况

### 终端区域
- **连接指示器**: 显示与 OpenClaw 的连接状态
- **命令历史**: 显示执行过的命令和输出结果
- **快速命令**: 一行按钮提供常用的命令快捷方式
- **输入区域**: 输入自定义命令的地方

### 动画效果
- **声波动画**: 聊天输入时的波浪形动画
- **状态动画**: 各种状态卡片的平滑过渡动画

## 故障排除

### 应用启动问题
1. **检查依赖**: 确保系统满足最低要求
2. **检查 OpenClaw**: 确保 OpenClaw CLI 已正确安装且在 PATH 中
3. **权限问题**: 在某些系统上可能需要管理员权限

### 连接问题
1. **CLI 优先**: 应用会优先使用原生 CLI，如果没有连接问题
2. **WebSocket 回退**: 如果 CLI 不可用，应用会自动回退到 WebSocket
3. **权限验证**: 某些命令可能需要特定的权限

### 性能问题
1. **系统资源**: 检查系统是否有足够的内存和 CPU
2. **后台进程**: 关闭不必要的后台进程
3. **更新应用**: 使用最新版本的应用

## 高级功能

1. **原生 CLI**: 在 Tauri 环境中直接执行 `openclaw` 命令

## 常见问题

**Q: 为什么在桌面上看不到我的 OpenClaw 服务？**
A: 确保 OpenClaw CLI 已正确安装，并且应用有权限访问它。应用会优先使用原生 CLI 而不是网络连接。

**Q: 终端命令没有响应怎么办？**
A: 检查 OpenClaw 服务是否正在运行，并确认应用有足够的权限执行命令。

**Q: 应用可以离线使用吗？**
A: 部分功能可以在离线状态下使用，特别是那些利用原生 CLI 的功能。
